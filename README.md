# Gosei_Test

1. Project Structure
  - api: .net core api 3.0
  - spa: Single page application using Angular 9

2. How to run
  - api: 
    + install .net core >3.0
    + go to 'api folder' and run 'dotnet run' to start service
  - spa:
    + install lastest angular cli
    + go to 'spa folder' run 'npm install'
    + run 'ng serve'
